Como Usar o Sistema

Este guia irá ajudá-lo a utilizar todas as funcionalidades do Sistema de Gestão Escolar.

1. Fazer Login

Acesse o sistema usando suas credenciais de usuário e senha.

2. Selecionar Unidade Escolar

Se você tem acesso a múltiplas unidades, selecione a unidade desejada.

3. Navegar pelo Dashboard

Use o menu de navegação para acessar as funcionalidades disponíveis para seu perfil.

Visualizar Notas
  • Acesse "Minhas Notas" no menu principal
  • Veja suas notas por disciplina
  • Consulte sua média geral
Entregar Trabalhos
  • Acesse "Meus Trabalhos" no menu
  • Clique em "Entregar" no trabalho desejado
  • Faça upload do arquivo e envie
Acompanhar Desempenho
  • Monitore suas notas no dashboard
  • Verifique trabalhos pendentes
  • Consulte frequência escolar

Gerenciar Disciplinas
  • Acesse "Minhas Disciplinas" no menu
  • Veja as turmas atribuídas
  • Consulte o cronograma de aulas
Lançar Notas
  • Acesse "Notas" no menu principal
  • Selecione a turma e disciplina
  • Digite as notas dos alunos
  • Salve as alterações
Criar Trabalhos
  • Acesse "Trabalhos" no menu
  • Clique em "Novo Trabalho"
  • Preencha as informações necessárias
  • Defina a data de entrega
Solicitar Recursos
  • Acesse "Requerimentos" no menu
  • Clique em "Novo Requerimento"
  • Descreva sua solicitação
  • Envie para a administração

Gestão de Alunos
  • Acesse "Gestão" > "Alunos" no menu
  • Cadastre novos alunos
  • Edite informações existentes
  • Matricule alunos em turmas
Gestão de Professores
  • Acesse "Gestão" > "Professores" no menu
  • Cadastre novos professores
  • Atribua disciplinas
  • Gerencie especializações
Gestão de Turmas
  • Acesse "Gestão" > "Turmas" no menu
  • Crie novas turmas
  • Atribua disciplinas às turmas
  • Gerencie matrículas
Análise de Requerimentos
  • Acesse "Requerimentos" no menu
  • Visualize solicitações pendentes
  • Analise e responda requerimentos
  • Gere relatórios de análise
