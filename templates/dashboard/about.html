Sobre o Sistema de Gestão Escolar

Visão Geral

O Sistema de Gestão Escolar é uma solução completa desenvolvida em Django para gerenciar instituições educacionais de forma eficiente e organizada.

Características Principais

  • Multi-tenant: Suporte a múltiplas unidades escolares
  • Controle de Acesso: Sistema RBAC (Role-Based Access Control)
  • API REST: Interface completa para integração
  • Autenticação JWT: Sistema seguro de tokens
  • Interface Responsiva: Funciona em qualquer dispositivo

Funcionalidades por Perfil

Administradores

Gestão completa de alunos, professores, turmas e disciplinas. Análise de requerimentos e geração de relatórios institucionais.

Professores

Gestão de disciplinas e turmas, lançamento de notas, criação de trabalhos e solicitação de recursos.

Alunos

Visualização de notas, entrega de trabalhos acadêmicos e acompanhamento do desempenho escolar.

Tecnologias Utilizadas

Backend
  • Django 4.2+ - Framework web principal
  • Django REST Framework - API REST
  • SimpleJWT - Autenticação via tokens
  • PostgreSQL/SQLite - Banco de dados
Frontend
  • Bootstrap 5 - Framework CSS
  • Bootstrap Icons - Ícones
  • JavaScript - Interatividade
  • Django Templates - Renderização
